### Running locally

First create a `.env` file and fill it with the variables described on the `.env.example` file.

```bash
  cp .env.example .env
```

Then, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
